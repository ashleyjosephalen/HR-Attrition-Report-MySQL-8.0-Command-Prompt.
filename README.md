# HR Attrition Analysis

## Overview

This project analyzes employee attrition data to identify key factors affecting employee retention and turnover.

## Features

* Data extraction using SQL
* Data Cleaning and EDA
* Analysis of salary, performance, and engagement
* Insight generation for HR decision-making

## Tech Stack

* SQL (MySQL)
* Python (Pandas, NumPy)
* Matplotlib, Seaborn

## Key Insights

* Identified departments with high attrition
* Analyzed correlation between salary and retention
* Highlighted employee engagement trends

## Database

* Designed relational database with 8 tables
* Used joins, subqueries, and aggregate functions

## How to Run

* Import dataset into MySQL
* Run SQL queries
* Perform analysis using Python
